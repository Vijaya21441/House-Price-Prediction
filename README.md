# 🏠 House Price Prediction

## 📌 Project Overview

This project predicts house prices using Machine Learning models like:

* Linear Regression
* Decision Tree
* Random Forest

## 📊 Features Used

* Area
* Bedrooms
* Bathrooms
* Stories
* Parking

## ⚙️ Technologies

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib, Seaborn

## 📈 Model Performance

* Linear Regression: High accuracy
* Random Forest: Best performance

## 💡 Features

* Price prediction
* Feature importance visualization
* Interactive sliders (ipywidgets)
* PDF report generation

## ▶️ How to Run

1. Open notebook in Jupyter / Colab
2. Run all cells
3. Use sliders to predict price

## 📂 Files

* `Housepriceprediction.ipynb`
* `house_price_model.pkl`
